void Qn::CorrectionHistogramBase::CopyTHnFDimension ( THnF *  hDest,
THnF *  hSource,
Int_t *  binsArray,
Int_t  dimension 
)
protected

Process a dimension in the copy of two THnF histograms process.

Source should not have channel/group structure (axis) while dest should. The channel/group involved in dest should be stored in binsArray[nVariables]. It is called in a recursive way until the number of event class variables (dimensions in the source) is exhausted then, the corresponding bins are copied.

Parameters
hDestthe histogram that will receive the copy
hSourcethe histogram to copy
binsArraythe array to build the bin numbers on each dimension
dimensionthe current dimension being handled
