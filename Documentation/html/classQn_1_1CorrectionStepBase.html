◆ QnCorrectionStepStatus

enum Qn::CorrectionStepBase::QnCorrectionStepStatus

The class of the id of the correction steps states.

Actually it is not a class because the C++ level of implementation. But full protection will be reached when were possible declaring it as a class.

When referring as "data being collected" means that the needed data for producing new correction parameters are being collected.

Enumerator
QCORRSTEP_calibration 

the correction step is in calibration mode collecting data

QCORRSTEP_apply 

the correction step is being applied

QCORRSTEP_applyCollect 

the correction step is being applied and data are being collected

QCORRSTEP_passive 

the correction step is waiting for external conditions fulfillment
